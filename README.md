# Firebase-GraphQl-Phone-Book
Build Phone Book Application to implement firebase and graphql in backend and react-redux in front-end

# Tech stack :
Back-end :
- Node.js
- express.js
- graphql
- Firebase Realtime Database

Front-end :
- React.js
- React-redux
